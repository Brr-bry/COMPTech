COMPUTER HARDWARE AND SOFTWARE

COMPUTER HARDWARE

Computer hardware are the parts that are required for a computer to function. These components are mainly composed of the central processing unit, memory, storage, terminals, and printers.
Examples:
  • Mouse
  • Keyboard
  • RAM Stick
  • Graphics Card

COMPUTER SOFTWARE

Software is a set of instructions, programs, or data that enables computers to perform specific tasks or functions. There are two main types of computer software; the system software, and the application software.

There are two main types of computer software:
  • System Software - is a type of software that helps a computer operate and manage its hardware. It acts as a base for other programs and ensures everything on the computer works properly. An example of system software is the operating system of the computer.
  • Application Software- includes programs that help users do specific tasks, like writing, browsing the web, or playing games. Unlike system software, which makes the computer work, application software helps users get things done.
