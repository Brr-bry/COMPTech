NUMBER SYSTEMS

WHAT ARE NUMBER SYSTEMS

A number system is a way of expressing numbers using a consistent set of symbols or digits. It provides a framework for representing and manipulating numbers in mathematics, computing, and everyday life.

DECIMAL NUMBER SYSTEM (BASE-10)

The decimal number system is what we humans use for our everyday lives. It is the number system that was taught to us since we were young.
Digits:
  • 010
  • 110
  • 210
  • 310
  • 410
  • 510
  • 610
  • 710
  • 810
  • 910
It is the conventional way we use numbers.
Examples:
  • 12010 = 7816
  • 10010 = 11001002

BINARY NUMBER SYSTEM (BASE-2)

Binary number system is an essential for computers. Binary digits can be interpretted as 1 for true and 0 for false.
Digits:
  • 02
  • 12
Examples:
  • 1012 = 5 10
  • 1001012 = 32 10
  • 102 = 210

OCTAL NUMBER SYSTEM (BASE-8)

The octal number system is used in computing to represent binary combinations in a more compact way. A combination of 3 binary digits can be presented in an octal number system.
Digits:
  • 08
  • 18
  • 28
  • 38
  • 48
  • 58
  • 68
  • 78
Example:
  • 158= 1310
  • 58= 1012
  • 108= 001 0002

HEXADECIMAL NUMBER SYSTEM (BASE-16)

The hexadecimal number system is also used in computing to write large binary combinations in a more compact way. The result of a combination of 4 binary digits is a digit from hexadecimal number system. Hexadecimal has 16 digits. It replaces digits greater than 9 into letters.
Digits:
  • 016
  • 116
  • 216
  • 316
  • 416
  • 516
  • 616
  • 716
  • 816
  • 916
  • A16
  • B16
  • C16
  • D16
  • E16
  • F16
Examples
  • F16 = 1510
  • 10A16 = 0001 0000 10102
  • 8B16 = 1000 10112
